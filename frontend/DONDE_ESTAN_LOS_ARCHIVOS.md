## 🔍 CÓMO ENCONTRAR LOS ARCHIVOS

### En Windows:
```
1. Abre Explorador de archivos (Windows + E)
2. Clica en "Descargas" en el menú izquierdo
3. Busca la carpeta "jinbu-app"
4. Doble clic para abrirla
```

### En Mac:
```
1. Abre Finder (Cmd + Spacebar, escribe "Finder")
2. Clica en "Descargas"
3. Busca la carpeta "jinbu-app"
4. Doble clic para abrirla
```

### En Linux:
```
1. Abre tu navegador de archivos
2. Ve a ~/Descargas/
3. Busca la carpeta "jinbu-app"
```

## 🚀 UNA VEZ DESCARGADOS

### Paso 1: Localiza la carpeta
```
Descargas/jinbu-app/
```

### Paso 2: Abre la terminal en esa carpeta

**Windows:**
```
1. En la carpeta jinbu-app
2. Haz clic derecho
3. "Abrir en Terminal" o "PowerShell"
```

**Mac/Linux:**
```
1. Abre Terminal
2. Escribe: cd ~/Descargas/jinbu-app
3. Presiona Enter
```

### Paso 3: Instala
```bash
npm install
```

### Paso 4: Ejecuta
```bash
npm run dev
```
